# Ring Desktop migration provenance

This directory was imported into the Eureka Assistant repository from:

- Repository: `https://github.com/IgniteTheSpark/ring-desktop.git`
- Base commit: `38efd6ea6924bd09a16d327f3441b4a13b8c7b90`
- Import date: `2026-07-14`

The import intentionally includes the source working tree's uncommitted M2
changes and new control, notification, vibration, and test modules. Local-only
files and generated state were excluded: `.git/`, `.venv/`, `.pytest_cache/`,
`__pycache__/`, `*.pyc`, `config.json`, and `.DS_Store`.

The former standalone checkout is not modified by this migration. From this
point onward, `ring-desktop/` in Eureka Assistant is the integration source of
truth for the Ring Demo.
