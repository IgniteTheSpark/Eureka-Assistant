# Eureka Assistant — 真机 QA 清单（TestFlight beta 前）

> 目的：在**全新构建**上系统验证最近的改动（刷新/可见性、时间解析、日历日视图、
> 事件创建、文件下线、通知点击、左滑删除）。
> ⚠️ **全新构建**：hot-reload **不会**应用 initState listener / lifecycle observer /
> ValueListenableBuilder 之类的结构改动。每轮 QA 前先 **hot-restart（按 `R`）或重跑
> `flutter run`**，否则你测的是旧逻辑（大量「刷新类」假 bug 都是这个原因）。

## 0. 构建前置
- [ ] `cd mobile && flutter run`（或重装 app），确认是**这次**的构建。
- [ ] 用一个**新账号**登录（每账号数据隔离）；后端 `/health` 返回 ok。

## 1. 刷新 / 可见性（核心回归区）
- [ ] **新建技能立即出现**：资产库 → ✨新技能 → 走完向导 → 返回，技能格**立刻**多出该技能。
- [ ] **手动快创**：底部 `+ 快创` → 选技能 → 填字段（含日期）→ 保存，不报 `Instance of 'DateTime'`；
      返回后该技能格计数 +1，且「最近」里出现。
- [ ] **agent 创建可见**：Agent 说「买牛奶」→ 待办 +1，资产库/日历「流」都出现。
- [ ] **闪念创建可见**：闪念说「刚刚花了 80 买咖啡」→ 整理完 → 资产库/日历同步、计数 +1。
- [ ] **计数器同步**：任何方式新建后，对应 container 数字立即 +1（不是停旧值）。
- [ ] **回前台刷新**：切到别的 app 几秒再回来 → 列表自动刷新。
- [ ] **删除后刷新**：删一条 → 列表与计数立即更新。

## 2. 时间解析（「刚刚」不再变 00:00）
- [ ] Agent/闪念「**宝宝刚刚喝了 150ml 奶**」→ 时间是**当前时刻（含时分）**，不是 00:00。
- [ ] 「**刚刚**花了 55 打车」→ expense 时间是当前时刻。
- [ ] 「**明天下午五点**开会」→ 日期=明天、时间=17:00。
- [ ] 任意自定义带时间字段技能再验一次「刚刚…」。

## 3. 日期不跨天（手动选日期）
- [ ] 快创一个带日期/时间的资产（如待办，截止 = **今天傍晚某时刻**）→ 保存 → 在日历/详情里
      看到的日期**还是今天**，不会跳到明天（旧 bug：选 6.4 存成 6.5）。

## 4. 日历 — 流 / 月 / 年 + 日视图（新）
- [ ] **流**：默认滚到「今天」附近；条目可点开详情；左滑可删。
- [ ] **月**：
  - [ ] 点一天 → 选中（底部 footer 列当天条目）。
  - [ ] **footer 高度固定**：选「忙的一天」vs「空闲的一天」，上面的月网格**不上下跳动**。
  - [ ] **再点同一天**（或点 footer 右上「日程 ›」）→ 打开**日视图**。
- [ ] **年**：12 宫格，点某月回到月视图。
- [ ] **日视图 · 列表（默认）**：
  - [ ] 当天全部条目按时间升序；**卡片等高**（有/无副标题大小一致）。
  - [ ] 点条目 → 详情弹层；**关掉弹层后条目不消失**（不闪空白）。
  - [ ] 左滑卡片 → 确认 → 删除，干净消失。
- [ ] **日视图 · 日程（右上 toggle）**：
  - [ ] **网格只画事件（日程）**：待办/记账/笔记/名片**不**出现在小时网格里。
  - [ ] **顶部「今日捕捉」按类型分 tab**（记账/读书笔记/名片/待办…），点 tab 切换。
  - [ ] **切 tab 时网格不跳**：今日捕捉固定 3 行高；少于 3 行留空、多于 3 行内部滚。
  - [ ] 「今日捕捉」和「日程安排」两个小标题**样式一致**。
  - [ ] 今天有红色「now」横线；全天事件走顶部 chip。
  - [ ] 今日捕捉里的卡片可左滑删除；网格事件块点开 → 详情里删除。

## 5. 创建表单
- [ ] **事件**：快创 → 事件 → 填标题；
  - [ ] **关「全天」**：要能选**开始 + 结束**时间，结束须晚于开始；保存成功（不再报
        `event missing time span` 400）。
  - [ ] **开「全天」**：结束时间隐藏，保存成功。
  - [ ] 有**描述**多行输入框，能填能存。
- [ ] **技能创建**：向导能生成草稿、预览、创建成功。

## 6. 卡片左滑删除（全类型一致）
- [ ] 资产库「最近」、技能详情页、日历流、日视图列表/今日捕捉、chat 内联卡 —— 都能左滑删除。
- [ ] **不可删**：闪念捕捉(input_turn)、task、无 id 的预构卡 → 左滑无效（无误删）。

## 7. 资产库
- [ ] **常驻只有 3 个**：事件 / 名片 / 外部。**没有「文件」**（已下线）。
- [ ] **「最近」跨类型**：刚建的**名片、事件**也出现在最近列表（不只是 asset）。
- [ ] 技能格计数与点进去的实际条数一致。

## 8. 技能删除
- [ ] 自建非受保护技能：详情页右上 🗑 → 有资产时提示「会同时删 N 条」→ 删除 → 技能格少掉它。
- [ ] 受保护内建技能（todo/idea/expense/notes/misc）**不显示** 🗑。

## 9. 通知（点击打开 / 跳转）
- [ ] **待办到期提醒**（「15 分钟后到期: …」）点击 → 弹出该**待办**详情层。
- [ ] **事件提醒**（「15 分钟后: …」）点击 → 弹出该**事件**详情层。
- [ ] **闪念完成**（⚡）点击 → 跳转该**会话**页。
- [ ] 点过的通知标记已读；「全部已读」清空红点。

## 10. 会话 / header / 登录
- [ ] 新开 Agent 但**不输入**就退出 → **不**产生空会话；有输入的会话标题**可读**（不是统一「Agent」）。
- [ ] 会话抽屉左滑可删会话。
- [ ] 全局 header 常驻（昼夜/通知/个人中心/设备），chat 路由里不出现；header 与内容间无大留白。
- [ ] 登出 → 登录页；重登只见自己数据；两账号互不可见。

---
### 出问题时的排查顺序
1. **先确认不是旧构建**：hot-restart / 重装后还复现吗？刷新类问题十之八九是热重载残留。
2. 后端 `docker logs eureka-assistant-backend-1` 看有没有 `tool_create_asset` /
   `index_asset_fields` 的 `TypeError`（多半是某技能 `queryable_fields` 形状不对）。
3. `curl` 直接打 `/api/skills`、`/api/assets?user_skill_name=<skill>`、`/api/timeline` ——
   返回了但 app 不显示 → 前端刷新/构建问题；没返回 → 后端/创建问题。
