## 0. 产品 · 受众 · 这屏要答什么

- **UReka** = 面向**普通人**(宝妈/保姆/老人/学生)的 AI 记录伙伴,陪伴宠物 **Reka**(蓝身金宝石,渲染器 `mobile/assets/js/mascot.js`)。深色 atmosphere + 品牌蓝 `#6F9EFF` + 低饱和 §8 八域色。
- **首页要答的不是"我今天记了啥"(那是回看,交给气泡池/日历),而是「下一步要做什么 + Reka 能帮我做什么」。** 是 app 的脸 = 每天打开的理由。
- **基调铁律(承全局)**:**高级 / 直观 / 有交互**;**不卡通玩具感**;**对老人尤其:默认那一屏必须一眼就懂、不靠"发现"**。

## 1. 层级(从下到上)

```
④ Reka 浮球(全局,不当主角)
③ 浮层 / bottom sheet(长按球的同类浮层、记录详情)
② 前景内容:【今日安排】⇄【Reka Offer】(全局二选一,各有 墙/Tinder 两模式)
① 气泡池(今天的捕捉,永远最底层背景)
   暖顶(早安 + 今日一览 · 吸收了晨报)挂在前景顶部
```

## 3. 两个前景屏(完全独立、各自全屏逻辑)

### 3.1 今日安排（events + todos + 有时间的习惯）
- **内容**:今天的日程、待办、**有惯常时间的习惯**(喝水 11 点)。**没时间的提醒/逾期不在这屏 → 去 Reka Offer。**
- **▦ 墙模式** = 卡片墙,**看全今天的安排**(滚动;可按时间序)。
- **⚡ Tinder 模式** = 一张张卡,**左右滑 = 上一个/下一个(纯浏览,不 dismiss/execute)**;焦点卡 = 下一件,带**每秒倒计时 + 进度**;滑完 → **↻ 回到当前**。
- **卡上按钮**:**事件 = 无提醒按钮**(自动提醒,只显被动「🔔 到点自动提醒」一行);**待办 = `完成 ✓` + `⏰ 延后` 两个独立按钮**,延后 = 卡上按钮(**不是滑**)→ popover **1 小时 / 明天 / 后天 / 自定义**(长按 = 精确),延到哪天就去哪天。

### 3.2 Reka Offer（Reka 能帮你做 / 你还没做）
- **内容**:**offer**(整理随记 / 消费总结 / 学习 quiz / 会前调研)+ **没固定时间的习惯提醒**(今天还没记单词)+ **逾期待办**。
- **▦ 墙模式** = 卡片墙,看全今天所有 offer。
- **⚡ Tinder 模式** = **左滑 ✕ 跳过 / 右滑 ✓ execute**;滑完 → **↻ 重新生成**(重发刚跳过的):
  - **offer → 右滑 = Reka 当场做**(一键出报告,**这是最爽的魔法一拍**)。
  - **习惯 → 右滑 = 开这条技能的快记**(你去记;喝水这种可一键记一杯)。
  - **左滑跳过 = 软「今天不想做」**:进通知 feed(留 14 天可翻回做)+ **压一天**(当天不再 offer,第二天条件仍在则重新 offer)。**不是「14 天不唠叨」**(那只挡 push peek)。详见 [§14.5a](14-proactive-reka.md)。
- **Tinder 拖拽时屏幕中央淡入大动作图标 + 标签**(Tinder 式):**蓝 = 浏览 / 绿 = 执行 / 红 = 跳过 / 琥珀 = 延后**;底部双按钮镜像。

### 3.3 切换
- **全局**:【今日安排】⇄【Reka Offer】—— 建议做成**整屏左右 swipe**(自然),顶部段控指示。
- **模式**:每屏一个小 **▦/⚡** 图标切 墙/Tinder(280ms 交叉淡入)。

### 3.4 气泡池（永远最底层背景 = 今天的捕捉）
- 一条记录 = 一颗气泡(**§8 域色填充 + 类型 emoji glyph**)。物理:无壁盒 + 倾斜重力 + 拖抛 + 新记录顶部 drop-in + 休眠去抖(参数见 prototype README)。
- **点球 → bottom sheet(单条详情)**。
- **长按球 → 首页开一个毛玻璃浮层**:把**这个类型今天所有卡**铺出来 + **同时高亮池里所有同类球**;点任意卡 → bottom sheet。(= 高级版"看这类今天记了啥",取代旧的类型 chips。)

## 4. 卡片:按类型各长各样（反枯燥的关键）

每张 = 一个小交互,**不是统一文字行**;**域色单点 + 类型 emoji**(承 §D 颜色收敛:卡本体单色 + 一颗域色点):

| 类型 | 卡的样子 + 一键 |
|---|---|
| 事件 | 倒计时滴答 + 进度条 + 被动「🔔 到点自动提醒」(无按钮) + 在日历 |
| 待办 | `完成 ✓` + `⏰ 延后`(两按钮);勾选那一下要爽 |
| 逾期 | 温柔「拖了 2 天」+ 改期/完成(**不愧疚**) |
| 习惯/streak | 🔥 连续 3 天 + 「今天还没」+ 记一笔 |
| offer 整理/总结 | 💡✨ 「10 条随记」+ [帮我理一理]→一键出报告 |
| 跟进 | 「给妈妈买礼物 → 设提醒?」+ [设] |

> offer/习惯卡可带一个 **Reka emote/小气泡**(💡 等),让它是「Reka 递给你的」,不是冷待办。

## 5. Reka（全局浮球 · 不当主角）

- Reka = 浮球(已存在),角落微反应;**不居中、不做仪表盘**。
- **Reka 的"声音" = 统一对话气泡**(Kenney 形,见 [§9.2.0](09-pet.md)):菜单/peek/nudge/chat 都用**带尾巴指向球球的气泡**(9-slice 或重画 + **按日夜主题/aura 染色** + 毛玻璃芯)。
- **emote**(Kenney CC0,`mobile/assets/emotes/`):头顶 pop + 通知/offer 图标(idea/cash/?/!/Z/★…);**gentle-only:不摆怒/哭/碎心脸**。

## 6. 状态（必须齐）

- **暖顶**:清晨池空时撑场(早安 + 今日一览);白天退居其次。
- **今日安排空**:🌤️「今天没安排,随便记」。
- **Reka Offer 空**:暖空态「今天没有新建议,记点啥都行」+ 可一键切回安排(**别让用户切过去看到白板**)。
- **气泡池空**:🫧「今天还没记录」。
- **轻日子 / 满日子**:都要好看(单薄问题靠"暖顶 + 池背景 + 双屏"撑,不靠填空)。

## 7. 质感 / 动效（高级 / 直观 / 有交互）

- **高级**:克制、材质/光/深度,不卡通玩具感。
- **直观**:**默认屏一眼懂**(老人也行);墙/Tinder/长按 = 进阶发现,不挡基本使用。
- **有交互**:Tinder 滑(物理回弹)· 长按球 → 毛玻璃浮层铺开(同类卡 + 球高亮)· 倒计时滴答 · 气泡 拖/倾斜/drop-in · offer 右滑→一键生成的爽感。

## 8. 已定（产品）+ design's call

**✅ 已定**：
- **默认进 = 今日安排 · Tinder**（打开就知道下一步,老人一眼懂)。
- **chips 砍掉** —— **不做类型 chips**;"看同类今天记了啥"**统一走「长按球 → 同类毛玻璃浮层」**(§3.4)。
- **域色 = B「潮汐」低饱和 8 板,已锁 + 解耦**(真源 `domainColor()`,[§8.3](08-domain-system.md);别硬编 hex,调函数)。
- **offer dismiss = 软「今天不想做」+ 压一天**(非 14 天不唠叨);**↻ 重新生成 = 重发刚跳过的**;**增量水位线**只总结新批([§14.5a](14-proactive-reka.md))。
- **⚡闪念 = 打开当日闪念 session**(非开 chat)。
- **天气 = 和风 QWeather + IP 城市**(server-side key,不弹定位权限)。
- **待办延后 = 卡上按钮(1h/明天/后天/自定义)**;**事件无提醒按钮(自动)**。

**design's call(你来定/探)**：全局切换形态(整屏 swipe vs 顶部段控)、墙的版式、长按浮层的揭示动效、暖顶的呈现、Tinder 滑动手感、卡片各类型美术。

## 9. 别做

- 不做旧原型的 **3 图表 dashboard**、**swipe 卡流**、**独立晨报页**(全废,本版取代)。
- Reka **不当首页主角**(只浮球)。
- 别堆色 —— 颜色只代表 §8 域(卡单色 + 域点),别 8 色 + 卡底色混成一锅。
- 别让默认屏需要"学"(老人)。
